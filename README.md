## Installation
```bash
npm install [--save] cleanr
```

## Usage
```javascript
const clean = require('cleanr');

const my-object = {...}

console.log(clean(my-object));
```
> This will log `my-object` without the empty parameters to the console.

## What is `empty`
  - {}
  - []
  - '' & ' '
  - undefined
